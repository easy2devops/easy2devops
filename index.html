Goploy - 自动化部署工具

Goploy是一个简单易用代码部署工具!

支持各种web代码发布，PHP，Python，JAVA等代码的发布、回滚

配置简单、功能完善、界面流畅、开箱即用

完整的安装指引，上手无难度

Demo Github Gitee

特性

  • Go语言开发，编译简单、运行高效
  • Web界面访问，交互友好
  • 权限模型灵活自由
  • Git、SVN仓库支持
  • 部署Hook支持，可扩展性强
  • 完善的上线工作流
  • 构建通知机制
  • 服务器监控
